
# BR-2: Order Status & Delivery Performance

## Business Objective
Analyze daily order status and delivery efficiency to support
SLA monitoring and operational performance tracking.

## Grain
**One row per DAY per ORDER STATUS**

## Final Output Table
brazillian_e_commerce.final.order_delivery_performance

## Key Metrics
- Total orders by status
- Average delivery days
- Delayed orders beyond SLA

## Use Cases
- Delivery performance monitoring
- SLA compliance analysis
- Order lifecycle tracking
- Operational dashboards

## Notes
This is a Gold-layer derived aggregate table optimized for
operational and logistics reporting.

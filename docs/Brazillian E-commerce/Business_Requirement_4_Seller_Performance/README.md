
# BR-4: Seller Performance

## Business Objective
Evaluate seller efficiency and contribution by analyzing revenue,
delivery delays, and customer review feedback.

## Grain
**One row per SELLER**

## Final Output Table
brazillian_e_commerce.final.seller_performance

## Key Metrics
- Total orders handled by seller
- Total revenue generated
- Average review score
- Number of delayed orders
- Delay rate per seller

## Use Cases
- Seller performance monitoring
- SLA and reliability tracking
- Marketplace quality analysis
- Seller ranking and evaluation

## Notes
This is a Gold-layer derived aggregate table built for analytics
and operational insights.

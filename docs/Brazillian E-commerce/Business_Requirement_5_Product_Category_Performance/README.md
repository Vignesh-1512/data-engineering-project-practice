
# BR-5: Product Category Performance

## Business Objective
Analyze sales trends and revenue contribution across product categories
to identify top-performing categories and growth patterns.

## Grain
**One row per DAY per PRODUCT CATEGORY**

## Final Output Table
brazillian_e_commerce.final.product_category_performance

## Key Metrics
- Total revenue by product category
- Total number of orders
- Average order value (AOV)

## Use Cases
- Category performance monitoring
- Merchandising and assortment planning
- Revenue trend analysis
- Executive dashboards

## Notes
This is a Gold-layer derived aggregate table optimized for BI
and analytical reporting.

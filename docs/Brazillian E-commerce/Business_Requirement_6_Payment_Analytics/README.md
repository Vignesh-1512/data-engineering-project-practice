
# BR-6: Payment Analytics

## Business Objective
Analyze customer payment behavior and payment method usage to
understand transaction trends and preferences.

## Grain
**One row per DAY per PAYMENT TYPE**

## Final Output Table
brazillian_e_commerce.final.payment_analytics

## Key Metrics
- Total orders per payment method
- Total payment value
- Average payment value

## Use Cases
- Payment method optimization
- Financial reconciliation
- Customer payment preference analysis
- Trend and growth analysis

## Notes
This is a Gold-layer derived aggregate table built to support
financial and analytical reporting.
